El box modelo en accion

El box model afecta el espacio y diseño de cada elemento según su display.

Display:Block

Cada elemento de tipo lock ocupa el ancho completo y se aplilan uno debajo del otro.

Display:Inline

Los elementos inline se alinean horizontalmente como texto

El margin horizontal (izquierda y derecha) funciona, pero el vertical (arriba y abajo) no afecta el flujo del documento. Es decir, se ignoran las propiedades.

Display: Inline-block

Los elementos inline-block se alinean horizonatalmente como inline pero permiten usar width height, border y margin como los elementos de display-block.
